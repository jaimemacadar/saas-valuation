## Glossário & Conceitos de Domínio

**SaaS Valuation**: Aplicação web para modelar, analisar e avaliar empresas SaaS usando dados financeiros e operacionais.

**Supabase**: Plataforma Backend-as-a-Service que fornece autenticação, banco de dados e armazenamento.

**Modelo de Valuation**: Conjunto de premissas e cálculos usados para estimar o valor de uma empresa SaaS.

**DRE (Demonstração do Resultado do Exercício)**: Relatório financeiro que mostra receitas, despesas e lucro.

**Balanço Patrimonial**: Demonstração financeira que mostra ativos, passivos e patrimônio líquido.

**FCFF (Fluxo de Caixa Livre para a Firma)**: Fluxo de caixa disponível para todos os provedores de capital após despesas operacionais e investimentos.

**WACC (Custo Médio Ponderado de Capital)**: Taxa média que uma empresa espera pagar para financiar seus ativos.

**Análise de Sensibilidade**: Técnica para determinar como diferentes valores de uma entrada afetam um determinado resultado.
